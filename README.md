# Grubby

Features/Use Cases
- Add/Edit/Remove recipe(s) with ingredients
- Create grocery lists by adding Recipes (compile ingredients from recipes)
- Log calories & macros

## Startup

ruby version 3.3.0  
rails version 7.0.8.1

```
bundle install
bin/rails db:migrate
bin/rails s
bin/rails tailwindcss:watch
```
